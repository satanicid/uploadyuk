Dokumentasi API UploadYuk

Integrasikan layanan penyimpanan media UploadYuk ke dalam aplikasi Anda.

Unggah Media

Gunakan endpoint ini untuk mengirimkan file secara langsung ke server kami.

POST /api/v1/upload

Parameter Form-Data

Key Tipe Deskripsi
media File File gambar, video, atau dokumen yang akan diunggah.

Contoh Penggunaan cURL

curl -F "media=@zack_ganteng.jpg" https://uploadyuk.web.id/api/v1/upload

Respons Sukses

{
  "id": 17154321000,
  "originalName": "zack_ganteng.jpg",
  "generatedName": "a1b2c.jpg",
  "url": "https://uploadyuk.web.id/a1b2c.jpg"
}

Catatan Teknis

  • Ukuran file maksimum bergantung pada konfigurasi server.
  • File yang diunggah akan diberikan nama acak unik.
  • Ekstensi file asli akan tetap dipertahankan.
